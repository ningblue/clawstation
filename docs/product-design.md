| **会话管理** | 创建、删除、重命名对话会话 | P0 |
| **OpenClaw集成** | 内嵌OpenClaw引擎，提供AI能力 | P0 |

| **主题切换** | 支持明亮/暗黑主题 | P0 |

## 2. 模型管理功能设计

### 2.1 服务商配置架构

基于OpenClaw的模型配置系统，ClawStation采用以下架构：

```
┌─────────────────────────────────────────────────────────────┐
│                    ClawStation 模型管理                      │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │   服务商层    │  │   模型层      │  │   配置层      │      │
│  │  Providers   │  │   Models     │  │   Config     │      │
│  └──────┬───────┘  └──────┬───────┘  └──────┬───────┘      │
│         │                 │                 │              │
│    ┌────▼─────────────────▼─────────────────▼────┐         │
│    │           OpenClaw Config Manager           │         │
│    │     (providers + models.json + auth)        │         │
│    └──────────────────┬──────────────────────────┘         │
│                       │                                     │
│              ┌────────▼─────────┐                          │
│              │   OpenClaw 引擎   │                          │
│              │  (Gateway + Agents)│                          │
│              └──────────────────┘                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 支持的服务商

根据OpenClaw配置，ClawStation支持以下AI服务商：

| 服务商 | 标识符 | 认证方式 | 默认模型 |
|-------|--------|---------|---------|
| Anthropic | `anthropic` | API Key | claude-sonnet-4-6 |
| OpenAI | `openai` | API Key | gpt-4o |
| Amazon Bedrock | `amazon-bedrock` | AWS Credentials | - |
| GitHub Copilot | `github-copilot` | GitHub Token | - |
| Google | `google` | API Key | gemini-pro |
| Azure OpenAI | `azure-openai` | API Key + Endpoint | - |

### 2.3 模型选择流程

```
用户操作流程:

1. 打开设置 → AI模型
   │
   ▼
2. 选择服务商 (Provider)
   │   - 从OpenClaw配置动态加载可用服务商列表
   │   - 显示已配置/未配置状态
   │
   ▼
3. 输入API Key
   │   - 验证API Key有效性
   │   - 安全加密存储
   │
   ▼
4. 选择模型 (Model)
   │   - 根据服务商动态加载可用模型
   │   - 显示模型信息（上下文窗口、最大token等）
   │
   ▼
5. 配置模型参数 (可选)
   │   - Temperature (0-2)
   │   - Max Tokens
   │   - Top P
   │
   ▼
6. 保存配置
   │   - 写入OpenClaw配置
   │   - 刷新模型列表
   │   - 应用新配置
```

### 2.4 配置存储结构

```typescript
// 服务商配置
interface ProviderConfig {
  baseUrl?: string;        // 自定义API端点
  api?: string;            // API版本/路径
  models?: ModelConfig[];  // 模型列表
}

// 模型配置
interface ModelConfig {
  id: string;              // 模型ID (如 anthropic/claude-sonnet-4-6)
  name: string;            // 显示名称
  contextWindow?: number;  // 上下文窗口大小
  maxTokens?: number;      // 最大输出token数
  input?: {
    cost?: number;         // 输入成本 (每1K tokens)
  };
  reasoning?: {
    cost?: number;         // 推理成本
  };
}

// 认证配置
interface AuthProfile {
  provider: string;        // 服务商标识
  apiKey: string;          // API密钥（加密存储）
  baseUrl?: string;        // 自定义端点
}
```

### 2.5 UI设计要点

**服务商选择器：**
- 下拉列表显示所有可用服务商
- 已配置的服务商显示绿色勾选标记
- 悬停显示服务商描述和官网链接

**API Key输入：**
- 密码输入框，支持显示/隐藏
- 实时验证按钮
- 安全提示："API Key仅存储在本地"

**模型选择器：**
- 级联选择：服务商 → 模型
- 显示模型卡片信息：
  - 模型名称和描述
  - 上下文窗口大小
  - 成本估算
  - 能力标签（代码、创意、分析等）

**参数调节：**
- Temperature滑块 (0-2，步进0.1)
- Max Tokens数字输入
- 恢复默认值按钮

## 3. 对话页面交互改进方案

### 3.1 当前界面分析

当前界面采用经典的三栏布局：
- 左侧：会话列表
- 中间：对话区域
- 顶部：状态栏和工具栏

### 3.2 交互改进方案

#### 3.2.1 输入区域增强

**当前问题：**
- 输入框功能单一
- 缺乏快捷操作

**改进方案：**

```
┌─────────────────────────────────────────────────────────────┐
│  快捷提示词栏 (可折叠)                                        │
│  [👋 自我介绍] [🐍 Python代码] [📝 内容总结] [🌐 翻译] [+ 自定义] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                     │   │
│  │  输入消息...                                         │   │
│  │                                                     │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐            ┌──────┐   │
│  │ 📎 附件  │ │ 🎙️ 语音  │ │ ⚙️ 参数  │            │ 发送 ➤ │   │
│  └─────────┘ └─────────┘ └─────────┘            └──────┘   │
└─────────────────────────────────────────────────────────────┘
```

**新增功能：**
1. **附件按钮** - 支持上传文件（P2）
2. **语音输入** - 点击开始语音转文字（P2）
3. **参数快捷调整** - 快速修改temperature等参数
4. **自定义快捷提示词** - 用户可添加自己的提示词模板

#### 3.2.2 消息展示优化

**代码块增强：**
```
┌─────────────────────────────────────────────────────────────┐
│  📋 复制  │  文件名: example.js  │  语言: JavaScript        │
├─────────────────────────────────────────────────────────────┤
│  1  │ function hello() {                                      │
│  2  │   console.log("Hello, ClawStation!");                   │
│  3  │ }                                                       │
└─────────────────────────────────────────────────────────────┘
```

- 语法高亮（使用highlight.js或prism.js）
- 行号显示
- 一键复制代码
- 代码折叠（长代码块）

**消息操作菜单：**
- 复制消息内容
- 重新生成回复
- 编辑消息（用户消息）
- 删除消息
- 引用回复

#### 3.2.3 会话列表改进

**新增功能：**
1. **会话搜索** - 快速查找历史会话
2. **会话分组** - 按日期分组（今天、昨天、上周、更早）
3. **会话固定** - 重要会话置顶
4. **批量操作** - 多选删除/导出

**视觉优化：**
- 显示会话最后一条消息预览
- 显示未读消息数徽章
- 拖拽排序

#### 3.2.4 模型切换快捷入口

在对话页面顶部添加模型切换器：

```
┌─────────────────────────────────────────────────────────────┐
│ ClawStation                                    [🔧 设置]     │
│                                                [❓ 帮助]     │
├─────────────────────────────────────────────────────────────┤
│  🤖 当前模型: Claude Sonnet 4.6  ▼  │  ⚡ 引擎运行中  ●       │
└─────────────────────────────────────────────────────────────┘
```

点击模型名称可快速切换模型，无需进入设置页面。

#### 3.2.5 流式响应优化

**打字机效果：**
- 逐字显示AI回复内容
- 光标闪烁指示正在生成
- 支持暂停/继续生成

**思考过程展示：**
```
🤔 正在思考...
   分析用户需求...
   检索相关知识...
   生成回复内容...

✅ 回复完成
```

### 3.3 响应式布局改进

**移动端适配：**
- 侧边栏可滑动收起
- 底部输入框固定
- 全屏对话模式

**桌面端优化：**
- 支持调整侧边栏宽度

### 4.3 技术对接点

```typescript
// OpenClaw Service API 对接
interface OpenClawIntegration {
  // 引擎管理
  startEngine(): Promise<void>;
  stopEngine(): Promise<void>;
  getEngineStatus(): Promise<EngineStatus>;

  // 模型管理
  getAvailableModels(): Promise<Model[]>;
  getProviders(): Promise<Provider[]>;

  // 对话
```

### 4.4 配置同步策略

```
ClawStation UI ──► OpenClaw Config Manager ──► ~/.clawstation/
                      │                          ├── config.yaml
                      │                          ├── agents/
                      │                          │   └── default/
                      │                          │       ├── models.json
                      │                          │       └── auth.json
                      │                          └── gateway.json
                      │
                      ▼
               OpenClaw Gateway
               (port 18791)
```

**同步规则：**
1. ClawStation修改配置 → 写入ConfigManager → 持久化到文件
2. Gateway自动监听配置变化（热重载）
3. 启动时从配置文件加载初始状态
